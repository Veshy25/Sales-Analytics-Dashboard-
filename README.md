# Sales Analytics Dashboard

A Power BI dashboard built to analyse B2B sales performance across 11 Southeast Asian markets, covering the period April 2025 – January 2026. The project simulates a realistic sales environment to demonstrate end-to-end business intelligence skills — from raw data to actionable insights.

---

## Project Objectives

- Apply core Power BI features from data import to dashboard publishing
- Clean and transform data using Power Query
- Build custom DAX calculated columns and measures for profitability analysis
- Design a professional, visually consistent dashboard using a custom Blue & White theme
- Derive meaningful business insights from raw sales data

---

## Dataset Overview

> **Note:** The dataset is AI-generated (via Claude by Anthropic) to simulate a realistic Southeast Asian B2B sales environment. It is intended for learning and portfolio purposes only.

| Property | Details |
|---|---|
| Source | AI-Generated (Claude by Anthropic) |
| Format | Microsoft Excel (.xlsx) |
| Rows | 1,467 order records |
| Columns | 13 fields |
| Time Period | April 2025 – January 2026 |
| Countries | 11 Southeast Asian markets |
| Product Categories | Electronics, Office Equipment, Packaging |

### Data Fields

| Column | Description |
|---|---|
| Order ID | Unique order identifier |
| Month | Order month (date format) |
| Country | Destination country |
| Customer Name | Buyer/client name |
| Product Category | High-level product grouping |
| Product Name | Specific product sold |
| Quantity | Units sold |
| UOM | Unit of measurement |
| Base Price | Price per unit before costs |
| Amount in LC | Total revenue in local currency |
| Total Packing Cost | Packaging cost per order |
| Total Freight | Freight/shipping cost per order |
| Variable Cost Rs per Unit | Variable manufacturing cost per unit |
